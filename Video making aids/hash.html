No server needed Runs in browser SHA-256 Dateless logic

Generate Hash

How to use this with pwd.js

  1. Enter the password you want to protect the page with.
  2. Generate the SHA-256 hash.
  3. Copy the generated line.
  4. Paste it into your pwd.js file.

Compatible logic

Your current pwd.js is now dateless. It simply hashes the entered password and compares it with the stored PASSWORD_HASH.

var PASSWORD_HASH = "paste_hash_here";

What gets hashed

Only the raw password is hashed now. No date is added. No rotation formula is used.

sha256(password)

Important warning

This is still client-side protection only.

It is suitable for lightweight lesson gating on static sites, but it is not true server-side security.
